โปรแกรมช่วยคำนวณด้าน electronics
================================

วิธีติดตั้ง (installation)
~~~~~~~~~~~~~~~~~~~~~~~~~~

เปิด CMD / Terminal

.. code:: python

    pip install electronics

วิธีใช้
~~~~~~~

เปิด IDLE ขึ้นมาแล้วพิมพ์...

.. code:: python

    import electronics

เปิด cmd / terminal ขึ้นมาแล้วพิมพ์...

.. code:: python

    python -m electronics
